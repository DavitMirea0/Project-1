## Основные возможности

### 1. Главная страница (index.html)

- Краткая информация о студенте
- Анимированные прогресс-бары навыков (C#, HTML/CSS, JavaScript, Qt, Figma)
- Превью основных проектов с переходом на детальную страницу
- Кнопка скачивания резюме

### 2. Страница проектов (projects.html)

- Каталог всех учебных и личных проектов
- Фильтрация проектов по технологиям. Пример:C#, Qt
- Модальные окна с детальным описанием проектов
- Галерея скриншотов для каждого проекта
- Навигация между изображениями (клавиши влево/вправо)
- Миниатюры изображений для быстрого переключения

### 3. Дневник (diary.html)

- Хронология учебного прогресса
- Записи о изучении различных технологий
- Визуализация текущих курсов с прогресс-барами
- Кнопка добавления новых записей

### 4. Контакты (contacts.html)

- Форма обратной связи с полями: имя, телефон, email, сообщение
- Контактная информация (email, GitHub, местоположение)
- Иконки для визуального представления контактов


## Функциональность JavaScript

### Анимация навыков
- Плавное заполнение прогресс-баров при загрузке страницы
- Intersection Observer для анимации при скролле
- Автоматическое отображение процентов после завершения анимации

### Система проектов
- Динамическое открытие модальных окон с информацией о проектах
- Фильтрация проектов по категориям
- URL параметры для прямых ссылок на проекты
- Галерея с навигацией и миниатюрами

### Открытие резюме
- Функция `openResume()` открывает изображение резюме в новой вкладке
